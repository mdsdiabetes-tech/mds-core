# \# MDS Product Model

# 

# \## Purpose

# 

# Every product in MDS Core is a knowledge object.

# 

# A product is more than something that can be purchased.

# 

# It is connected to people, diseases, medications, devices, guides, videos, FAQs, and recommendations.

# 

# \---

# 

# \## Product Categories

# 

# \- Identity

# \- Commerce

# \- Medical Information

# \- Compatibility

# \- Storage

# \- Travel

# \- Educational Content

# \- Related Products

# \- Alternatives

# \- Accessories

# \- Reviews

# \- AI Summary

# \- SEO

# \- Analytics

# 

# \---

# 

# \## Principle

# 

# A single product record should provide enough information to power:

# 

# \- Product pages

# \- Search

# \- AI

# \- Recommendations

# \- Comparisons

# \- Bundles

# \- Articles

# \- FAQs

# \- Structured Data
